## How to Use

These are Claude Skills ‚Äî custom instructions that turn Claude into specialized tools.

**Web or Desktop App:**
1. Download the `.md` file for the skill you want
2. Go to Claude ‚Üí Settings ‚Üí Claude Skills
3. Upload the file
4. Start a new conversation and describe what you need

**CLI:**
1. Create a folder with the skill name (e.g., `blockchain-saas-evaluator/`)
2. Rename the file to `SKILL.md` and place it inside the folder
3. Add to your CLI skills directory
